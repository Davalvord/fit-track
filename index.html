Conexión con Google Sheets

Mantén tu hoja de cálculo perfectamente limpia y actualizada. Con esta función puedes volcar todos tus datos locales de golpe a Google Sheets para que Gemini tenga el contexto histórico más completo y ordenado.

MÉTODO 1

Volcado Manual (Copiar y Pegar)

Genera una tabla de datos completa compatible con Excel y Sheets. Solo debes pulsar el botón, abrir tu hoja de cálculo, vaciarla, situarte en la primera celda (A1) y pulsar **Pegar (Ctrl+V)**.

MÉTODO 2

Sincronización Automática (1-Clic)

Sube y sobrescribe automáticamente toda tu hoja de cálculo utilizando un pequeño script de Google.

¿Cómo configuro el script en mi hoja?

1. Abre tu hoja de cálculo en **Google Sheets**.

2. Ve a **Extensiones > Apps Script**.

3. Borra todo el código del editor y pega el siguiente script:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.clear();
  if (data && data.length > 0) {
    sheet.getRange(1, 1, data.length, data[0].length).setValues(data);
  }
  return ContentService.createTextOutput(JSON.stringify({status: "success"}))
    .setMimeType(ContentService.MimeType.JSON);
}

4. Pulsa en **Implementar > Nueva implementación**.

5. Selecciona el tipo **Aplicación web**. Configura: *Ejecutar como:* Tú, *Quién tiene acceso:* Cualquier persona.

6. Pulsa **Implementar**, autoriza los permisos y copia la **URL de la aplicación web** para pegarla arriba.
