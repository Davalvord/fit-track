Conexión con Google Sheets

Mantén tu hoja de cálculo perfectamente limpia y actualizada de forma bidireccional. Puedes volcar tus datos locales a Google Sheets para que Gemini tenga el contexto histórico más completo, o recuperar toda la información en un móvil nuevo al instante.

Sincronización Cloud

Sincronización Bidireccional Automática

Introduce tu enlace de Apps Script para subir o descargar toda tu base de datos histórica.

¿Cómo configuro el script bidireccional en mi hoja?

1. Abre tu hoja de cálculo en **Google Sheets**.

2. Ve a **Extensiones > Apps Script**.

3. Borra todo el código del editor y pega este script bidireccional completo:

function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getDataRange().getValues();
  return ContentService.createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.clear();
  if (data && data.length > 0) {
    sheet.getRange(1, 1, data.length, data[0].length).setValues(data);
  }
  return ContentService.createTextOutput(JSON.stringify({status: "success"}))
    .setMimeType(ContentService.MimeType.JSON);
}

4. Pulsa en **Implementar > Nueva implementación**.

5. Selecciona el tipo **Aplicación web**. Configura: *Ejecutar como:* Tú, *Quién tiene acceso:* Cualquier persona.

6. Pulsa **Implementar**, autoriza los permisos y copia la **URL de la aplicación web** para pegarla arriba.
